W3C and HTML and CSS





W3C is an international community for developing web standards. W3C is led by CEO Jeffrey Jaffe and Director Tim Berners Lee. W3C works with the stakeholders,members and the public to make sure the standards developed are in consensus with everyone. CSS and HTML are some of the many standards developed by W3C

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are some of the most essential technologies for building a web page. HTML is the markup language used to display information on the web page. While HTML gives structure to the page, CSS is used to style the web page.

When a browser sends an http/https request to the host server, the html and css files will be returned to the browser. The browser will render this HTML and CSS pages to the user. Using HTML tags we can add headings, paragraphs, lists, links , images and input fields to a page making it more interactive and user friendly.

While HTML is used to provide structure to the page CSS is used to define the styles like color, position, height, width etc to the page. Multiple HTML pages can share the same CSS style by including the same CSS file.
