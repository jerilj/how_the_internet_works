DNS





Domain name system (DNS) translates the domain name we enter in the browser to the IP address of the server. If this is a frequently visited website, the browser might have the ip address stored for that domain name. If it is not present in the computer, a request will be sent to a list of DNS servers to look for the IP address. This request will continue until a domain name is found. Once it is found the IP address will be sent to the browser. The computer might store it for future release.

The DNS is distributed across the internet. Each DNS server only knows a limited amount of information. The first request from your browser goes to a resolver provided by the internet service provider. The resolver asks the root name server that has the information of the .com or .org part of the domain name. The root name server will return a pointer to another name server that can provide further information about the domain name.

When the owner of a website register a domain name, that record will be entered in DNS server. Once a browser receives the IP address from the DNS it can connect to the server that holds the website.
