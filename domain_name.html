Domain names and ICANN





Domain name is the name that is used to identify a computer/server connected to the network. When a user enters the domain name in the browser, the DNS server will look up the IP address for that domain name and forward request to that server hosting the website using the IP address. In order for this to work, a list of DNS servers must be made available with the list of domain names and the corresponding IP addresses

ICANN is a non-profit organization that maintains the IP address space for IPv4 and IPv6 addresses. ICANN is not controlled by one particular country. In order to connect to another computer/server on the internet a unique address is needed. ICANN maintains the list of unique addresses for this purpose. ICANN has a multi stakeholder model with representatives governments, educational institutions and private institutions etc. ICANN has a community driven policy that makes sure it represents everyone in the global internet community.

ICANN coordinates the domain name systems, IP address allocation, protocol parameter registry, root servers, top level domain names etc. ICANN ensures the security and standards of DNS. ICANN works with supporting organizations, advisory committees and technical advisory bodies to make sure the internet is a community driven.Registrars, gtld registry operators and ISP are also members of the ICANN stakeholder groups. The stakeholders can propose policies that will be adopted only with the consensus of other stakeholders.
